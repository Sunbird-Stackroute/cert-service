# Certificate Service

### Requirement
`Node >=12 <=13`
`Angular 7`

### Run locally,
1. `npm install`

2. `ng serve -o`

### Build for Production,
1. `npm i`

2. `npm run build`





### Incase of build error on production for `node-sass` use the below commands,
1. `npm cache clean --force`

2. `npm i --unsafe-perm node-sass`

3. Repeat steps from `Build for Production`
